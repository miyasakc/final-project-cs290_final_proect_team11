What is Monte Carlo's Simulation?

Monte Carlo simulation is a mathematical technique that utilizes random sampling from a range of input values to determine probability. The main focus of Monte Carlo simulation is to utilize the concept of random variability in order to solve problems. One use of the Monte Carlo simulation is to estimate the value of pi using the random sampling of the simulation to randomly generate various points in a circle within a square. Then divide the amount of points inside the circle by the points within the square and multiply the result by 4 in order to return an estimation of pi.
